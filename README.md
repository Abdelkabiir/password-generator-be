## Description

[nest](https://nestjs.com/). A progressive **Node.js** framework for building efficient and scalable server-side applications, heavily inspired by **Angular**.

## Installation
```bash
$ npm install
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
